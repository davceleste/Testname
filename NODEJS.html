Node.js es la plataforma de software más utilizada en este momento estando por encima en entornos de ejecución y lenguajes de programación como PHP y C a la vez que tiene un tiempo de ejecución (tiempo real) muy superior.

Proposito de Instalacion

Para Instalar Node Js justify, deberas:

  1. Descargar Node Js->  Descargar
  2. Abrir archivo de instalacion, seguir instrucciones (dar siguiente)
  3. Abrir Node Js
  4. Personalizar con las extenciones nesesarias
  5. Listo, ahora a trabajar con Node Js
