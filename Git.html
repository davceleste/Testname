Git es un software de control de versiones diseñado por Linus Torvalds, pensando en la eficiencia, la confiabilidad y compatibilidad del mantenimiento de versiones de aplicaciones

Proposito de Instalacion

Para Instalar Visual Studio Code, deberas:

  1. Descargar Git->  Descargar
  2. Abrir archivo de instalacion, seguir instrucciones (dar siguiente)
  3. Abrir Git
  4. Personalizar con las extenciones nesesarias
  5. Listo, ahora a trabajar con Git
